Flujo de datos

API REST + MongoDB Atlas + AWS EC2

Angular consume el mismo backend que Postman. Las operaciones GET, POST, PUT y DELETE llegan a la API REST, el backend procesa los datos y MongoDB Atlas conserva la información.

GET /misitio/health GET /misitio/gastos POST /misitio/gastos PUT /misitio/gastos/:id DELETE /misitio/gastos/:id GET /misitio/usuarios POST /misitio/usuarios PUT /misitio/usuarios/:id DELETE /misitio/usuarios/:id
